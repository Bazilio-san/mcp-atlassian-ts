После внесения изменений сам проверяй: 
Запускай
`npn run lint:fix`
и добейся отсутствия ошибок.
Запускай
`npn run cb`
и добейся отсутствия ошибок.
потом запускай в первой консоли 
`npm run start`
и отслеживай там логи.
во второй консоли запусти тест
`node tests\mcp\jira.js --tests=jira_get_project`

в первой консоли считывай логи 

для остановки процесса в первой консоли используй
`node scripts/kill-3000.js`
